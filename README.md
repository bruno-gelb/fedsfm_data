# fedsfm_data

Gather & visualize data from fedsfm terrorists and extremists list.

## stack

python 3.7, pipenv, plotly

## how to install

```bash
$ pipenv install
```

## how to run

```bash
$ pipenv shell
$ python fedsfm_data/run.py
```
